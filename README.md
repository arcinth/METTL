# Java Number Logic & Mettl Practice Programs

This repository contains a collection of **Java programs** created to practice and master **number-based logic questions** commonly asked in **Mettl, TCS, Accenture, Wipro, Zoho, and other placement exams**.

These problems focus on:
- Digits manipulation
- Prime numbers
- Fibonacci
- Counting logic
- Mathematical reasoning

All programs are written in **Java (JDK 7 compatible)** with clear logic and without using advanced libraries.
